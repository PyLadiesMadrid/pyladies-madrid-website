Hacktoberfest 2020

Estos son los proyectos que proponemos para el hacktoberfest de este año. Trabajaremos en grupo entre las PyLadies que se apunten a cada uno y nos organizaremos a través de Discord. Cada proyecto tiene una o varias mentoras que harán de apoyo y guía. Si quieres participar en uno, escribe en el canal de Discord correspondiente. Si aún no tienes acceso a Discord, escríbenos un mail.

Además, puedes echar un ojo a la web oficial del evento o a la web del evento organizado por Python España
